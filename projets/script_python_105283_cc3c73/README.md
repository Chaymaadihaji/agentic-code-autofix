# Fonctionnalité Principale de l'Application Web
==============================================

## Description de l'Application

Cette application web est conçue pour fournir une fonctionnalité principale. Les détails fonctionnels sont inclus à mesure.

## Fichier source associé

Ce README accompagne le fichier source associé [fichier_source.py](./fichier_source.py).

## Installation et Utilisation

L'installation et l'utilisation de cette application sont décrites ci-dessous.

### Installation de l'Environnement de Développement

1. Assurez-vous d'avoir Python (3.x) installé sur votre système.
2. Clonez le référentiel Git en utilisant `git clone`.
3. Installez les dépendances avec `pip install -r requirements.txt`.

### Exécution de l'Application

1. Exécutez le fichier source avec `python fichier_source.py`.
2. Accédez à la page d'accueil de l'application web à l'adresse `http://localhost:5000/`.

## Gestion d'Erreurs

Cette application inclut une gestion d'erreurs élémentaire pour les cas d'erreur suivants :

* Erreur lors de l'exécution du code source
* Erreur de requête HTTP

Les messages d'erreur sont affichés sous forme de fenêtre modale.

## Dépendances

Cette application repose sur les dépendances suivantes :

* Python 3.x
* Flask (framework web)
* Pandas (bibliothèque de data science)
* MatplotLib (bibliothèque de visualisation de données)
* Scikit-Learn (bibliothèque de machine learning)
