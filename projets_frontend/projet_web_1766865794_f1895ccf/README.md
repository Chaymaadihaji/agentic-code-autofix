# README.md

## Présentation du projet

Ce projet consiste à développer une application Android de météo qui prend en compte les fonctionnalités suivantes :

* Localisation
* Prévisions 7 jours
* Alertes
* Favoris villes

## Technologies utilisées

* Langage de programmation : Kotlin
* SDK Android
* API REST OpenWeather

## Informations de développement

* Version de l'API OpenWeather : v5
* Clé API : `YOUR_API_KEY` (remplacer par la clé API réelle)
* Version de l'application : 1.0

## Installation

1. Cloner le projet à l'aide de Git
2. Télécharger les dépendances nécessaires à l'aide de Gradle
3. Remplacer `YOUR_API_KEY` par la clé API réelle
4. Exécuter l'application à l'aide de l'émulateur Android ou d'un appareil réel

## Aperçu de l'interface utilisateur

L'application sera dotée d'une interface utilisateur moderne et intuitive, conçue en suivant les principes de la conception d'interface utilisateur mobile-first. Les composants UI attendus incluent :

* Cartes
* Formulaire
* Tableau
* Graphique

L'application disposera d'une barre de navigation pour accéder aux différentes fonctionnalités.

## Fonctionnalités clés

* **Localisation** : l'application permettra à l'utilisateur de localiser sa position géographique à l'aide de la géolocalisation.
* **Prévisions 7 jours** : l'application affichera les prévisions météorologiques pour les 7 prochains jours pour la ville sélectionnée.
* **Alertes** : l'application permettra à l'utilisateur de recevoir des alertes en cas de changement météorologique important.
* **Favoris villes** : l'application permettra à l'utilisateur d'enregistrer ses villes favorites pour accéder rapidement à leurs prévisions météorologiques.

## Commentaires

Cette application a été développée pour répondre aux besoins de la communauté météorologique. Elle est dotée d'une interface utilisateur moderne et intuitive, conçue en suivant les principes de la conception d'interface utilisateur mobile-first.
